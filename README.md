# Miscrit Console Game
Console based game written in Java

This is my first Java project it was created after ~1 month of learning Java to practice core programming concepts and object-oriented design.

## Features
- Turn-based combat system
- Elemental damage system
- Capture mechanics
- Leveling and XP system

## Technologies
- Java Core
- OOP principles

## What I learned
- Designing simple game logic
- Working with classes and inheritance
- Implementing combat and progression systems

## Future improvements
- Refactor code into better architecture
- Improve code readability and structure
- Add more game mechanics

## How to run
1. Clone the repository
2. Open in IDE
3. Run Main.java
